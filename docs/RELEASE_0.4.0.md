# TeenModo 0.4.0

- Direct model installation: `ENGINE ALIAS SOURCE`; no model YAML editing.
- Stable-diffusion.cpp bundled in the main archive.
- Generated manifests persist custom runtime profiles.
- Dynamic adapter loader registers modules safely before dataclass evaluation.
- User jobs are created from canonical examples with `teenmodo job create BASE ALIAS`.
- Controller and Compose versions aligned at 0.4.0.
- Interrupted CLI operations exit cleanly.
- Documentation split into quick start, model, job, backend, architecture, platform, and error guides.
