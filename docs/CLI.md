# CLI lifecycle

The current complete command reference is [COMMANDS.md](COMMANDS.md).

The primary workflow remains:

```bash
teenmodo doctor
teenmodo engine install ENGINE
teenmodo model install ENGINE MODEL [SOURCE]
teenmodo run ENGINE MODEL JOB
teenmodo report summary --engine ENGINE --model MODEL
```

Use `teenmodo paths` to locate reports, raw samples, generated files, and custom jobs.
